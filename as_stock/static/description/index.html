Product Last Prices Tab

Thanks to this module you can see the last prices of the product variant. Last price tab will be added next to the "General Information" tab.
Module adds:
- New tab to the product variants (Last Price)
- Tab contains a table with last purchase price, vendor name and date of purchase
- Tab contains a table with last sale price, customer name and date of sale

Easy to install and use

Module does not need any IT experience - simply install and restart te server. Module requires "Product Variants" activated (You can do this by entering the Sale/Settings/Product Variants). Last prices will be displayed if Sales Order or Purchase order is set to "Done"(Odoo 9) or "Lock" stage.
